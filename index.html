Tecnologías aprendidas

Hasta ahora, he aprendido sobre las siguientes tecnologías en el desarrollo web:

HTML
Es el lenguaje de marcado utilizado para estructurar el contenido de una página web.
CSS
Es el lenguaje de estilos utilizado para diseñar y dar formato a las páginas web.
JavaScript
Es un lenguaje de programación que se utiliza para agregar interactividad y dinamismo a las páginas web.

Etiquetas HTML

La siguiente lista de etiquetas son las que he aprendido de momento en mi curso de HTML

  1. <html>: Es la etiqueta raíz que envuelve todo el contenido de la página web.
  2. <head>: Contiene información meta sobre el documento, como el título y enlaces a hojas de estilo.
  3. <body>: Contiene el contenido visible de la página web, como texto, imágenes y enlaces.
  4. <h1> a <h6>: Son etiquetas de encabezado que se utilizan para estructurar el contenido en diferentes niveles de importancia.
  5. <p>: Se utiliza para definir un párrafo de texto.
  6. <a>: Se utiliza para crear enlaces a otras páginas web o recursos.
  7. <img>: Se utiliza para insertar imágenes en la página web.
